Question-1: What are the different ways to select an element in the DOM?

Answer:In the DOM (Document Object Model), elements can be selected using several methods in JavaScript. The most common ways are getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: In HTML, innerHTML, innerText, and textContent are properties of the DOM (Document Object Model). They allow you to read and update the content of HTML elements.

Question-3: What is event delegation in the DOM?

Answer:Event delegation in the DOM is a JavaScript design pattern that optimizes event handling by attaching a single event listener to a parent element to handle events that occur on its child elements. It works as: Event Bubbling, Single Event Listener, Identifying the Source, Event Handling.

Question-4: What is event bubbling in the DOM?

Answer: In the DOM, event bubbling is a default behavior where an event triggered on a child element also propagates up the DOM tree to its parent elements. This means if you click a button inside a form, the click event will not only trigger on the button but also on the form itself, and potentially on other ancestor elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer:JavaScript provides methods to dynamically create, add, and remove HTML elements.

  • Creating Elements: The document.createElement() method is used to create new HTML elements. It takes the tag name of the desired element as an argument.
  • Adding Elements: Elements can be added to the DOM using methods like appendChild() or insertBefore().
  • Removing Elements: Elements can be removed from the DOM using methods like removeChild() or remove().
